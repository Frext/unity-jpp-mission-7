## 🎯 Implemented Parts

* **Data Persistence:** Saves the last chosen theme color using JSON serialization, automatically pre-selecting it the next time the application is launched or **Load Color** button is pressed.
* **OOP Architecture:** Built on solid object-oriented pillars including **Inheritance** (specialized `TransporterUnit`), **Polymorphism** (overridden methods), **Abstraction** (base classes like `Unit` and `Building`), and **Encapsulation** (getters/setters).
* **Extensible Simulation:** Features different production speeds for resources, structural inventories, and targeted unit movement states.
